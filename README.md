# Cybersecurity Chatbot

A Windows Forms chatbot that educates users about cybersecurity topics like passwords, scams, phishing, and privacy.

## Features
- **GUI** – user-friendly interface with chat area and ASCII art
- **Keyword Recognition** – detects password, scam, privacy, phishing
- **Random Responses** – multiple tips per topic (keeps conversation fresh)
- **Conversation Flow** – handles "tell me more" and "another tip"
- **Memory** – remembers your name
- **Sentiment Detection** – responds empathetically to worried/frustrated/curious users
- **Voice Greeting** – speaks welcome message on startup
- **Error Handling** – default response for unrecognized input

## How to Run
- Open the solution in Visual Studio
- Build (Ctrl+Shift+B)
- Run (F5)

## Requirements
- .NET Framework 4.7.2 or higher
- Windows Forms

## Technologies Used
- C#
- Windows Forms
- System.Speech (for voice)
